# My Blog Website 📝

A simple **Blog Website** built using **Node.js, Express.js, EJS, and Bootstrap**.
This project allows users to create, view, edit, and delete blog posts.

It was developed while learning **Full Stack Web Development** to understand how a basic **CRUD (Create, Read, Update, Delete) application** works.

---

## 🚀 Features

* Create new blog posts
* View all blog posts
* View full blog content
* Edit existing blog posts
* Delete blog posts
* Dynamic routing using Express
* Server-side rendering using EJS
* Clean UI using Bootstrap and custom css

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/yourusername/my-blog-website.git
```

2. Navigate to the project folder

```
cd my-blog-website
```

3. Install dependencies

```
npm install
```

4. Run the server

```
node index.js
```

5. Open in browser

```
http://localhost:3000
```
